# AI Style Portfolio Website

Single-page personal portfolio built with plain HTML, CSS, and JavaScript. The layout is designed like an AI assistant interface instead of a traditional scrolling portfolio.

## Stack

- HTML
- CSS
- JavaScript

## Features

- AI-inspired dark interface with glass panels and fixed floating prompt navigation
- Hero section with greeting badge, premium profile image placeholder, and prompt-style actions
- Progressive section reveal with typing animation
- About, Skills, Projects, Experience, and Contact responses inside a chat-like stream
- Real project cards with technology badges, live website/app links, and NDA-protected work summary
- Responsive layout for desktop, tablet, and mobile
- Hover glow, fade-in, lift, and smooth scrolling interactions

## Files

- `index.html` - page structure, AI conversation content, and image placeholder blocks
- `style.css` - layout, theme, responsiveness, and animation styling
- `script.js` - typing effect, section reveal, smooth focus, and active prompt highlighting
- `README.md` - project overview

## Run

Open `index.html` in a browser.

## Notes

- The hero profile placeholder is intended for a `450x550px` image with a `4:5` ratio. PNG with transparent background is preferred.
- Project image placeholders are intended for `420x260px` previews with a `16:10` ratio.
- The Projects section now uses real portfolio entries, including public links and an NDA-protected card for confidential work.
- GitHub and email are wired from repo metadata.
- LinkedIn is left as a visible placeholder URL and should be replaced with the real profile link.
